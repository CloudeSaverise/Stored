Bô

Họ tên: Bùi Thanh Huy

Trường: THPT Nguyễn Trân

Lớp: 12A2

Giới thiệu bản thân

Điểm mạnh:

+ Thích đi sâu tìm hiểu những cái mới.

+ Tiếp thu khá tốt.

+ Thân thiện, dễ gần.

Điểm yếu:

+ Bị lười, thiếu tự tin.

+ Dễ sao nhãng.

Sở thích:

+ Âm nhạc.

+ Ăn uống.

+ Thể Thao.

Thói quen:

+ Hay nghe nhạc khi học bài, làm bài.

( Nên nhạc thì thuộc còn bài thì không! )
